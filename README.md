# Portfolio GitHub Pages

Portofolio statis profesional yang bisa langsung dipublish ke GitHub Pages.

## Cara pakai

1. Sesuaikan project, skill, dan pengalaman di `index.html`.
2. Pastikan repository GitHub bernama `hendraramadani.github.io` untuk publish sebagai situs utama.
3. Buka `index.html` di browser untuk melihat hasil lokal.
4. Push isi folder ini ke branch `main`.
5. Buka `https://hendraramadani.github.io`.

## SEO

Template ini sudah dilengkapi:

- Meta title dan meta description.
- Canonical URL.
- Open Graph untuk preview WhatsApp, LinkedIn, Facebook, dan Discord.
- Twitter Card.
- JSON-LD structured data untuk `Person` dan `WebSite`.
- `robots.txt`.
- `sitemap.xml`.
- Favicon SVG.

## Publish ke repository biasa

Kalau repository tidak bernama `hendraramadani.github.io`, buka GitHub repository:

- `Settings`
- `Pages`
- `Build and deployment`
- pilih `Deploy from a branch`
- branch `main`
- folder `/root`

GitHub akan memberi URL Pages untuk repository tersebut.
